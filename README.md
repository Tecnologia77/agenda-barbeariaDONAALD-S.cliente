# 💈 DONAALD'S Barbearia - Agendamento Online (Cliente)

Este é o módulo de interface com o cliente da **DONAALD'S Barbearia**. Uma solução web progressiva desenvolvida para que o cliente possa realizar agendamentos de forma rápida, intuitiva e em tempo real.

## 🚀 Acesse o Sistema
🔗 **Link:** https://tecnologia77.github.io/agenda-barbeariaDONAALD-S.cliente/

## ✨ Experiência do Usuário
* **Agendamento Inteligente:** Escolha de serviços, profissionais e horários disponíveis de forma dinâmica.
* **Confirmação Instantânea:** Integração com o banco de dados para garantir que não haja choque de horários.
* **Interface Mobile-First:** Desenvolvido para funcionar perfeitamente em qualquer smartphone, sem precisar baixar apps.

## 🛠️ Tecnologias
* **Frontend:** JavaScript (ES6+), HTML5 e CSS3.
* **Backend:** Firebase Realtime Database para sincronização de agenda.
* **Hospedagem:** Deploy em infraestrutura otimizada.
